# 案件ダッシュボード

エンジニア向けの案件管理アプリケーション。複数の案件を一元管理し、タスク、進捗、売上を効率的に追跡できます。

## 機能

### 📋 案件管理
- 案件の追加・編集・削除
- 顧客名、プラットフォーム（ココナラ、ランサーズ、直接案件など）、案件種別の管理
- ステータス管理（無料見積もり、有料見積もり、取引中、取引完了、保留中）
- 金額と納期の管理

### ✅ タスク管理
- チェックリスト形式のタスク管理
- タスクごとの期限設定
- 進捗率の自動計算

### 💡 提案・見積もり管理
- 追加提案の記録
- 見積もり金額と期限の管理
- 提案ステータスの追跡

### 📊 ダッシュボード
- 今月の確定売上表示
- 進行中案件数の表示
- 期限間近の案件アラート
- 見込み売上の計算

### 🔍 フィルタリング
- すべて / 今日 / 今週 / 今月
- 期限超過 / 今後の予定

### 💾 データ管理
- LocalStorageによる自動保存
- JSONファイルへの自動エクスポート（30秒ごと）
- JSONファイルからのインポート機能

## 技術スタック

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Zustand** (状態管理)
- **date-fns** (日付処理)

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番サーバーの起動
npm start
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方

### 案件の追加
1. 「+ 新規案件」ボタンをクリック
2. 案件情報を入力（案件名、顧客名、プラットフォーム、種別、金額、期限、ステータス）
3. 「保存」をクリック

### タスクの追加
1. 案件カードをクリックしてサイドパネルを開く
2. 「タスク」セクションの「+ 追加」ボタンをクリック
3. タスク名と期限を入力して追加

### データのエクスポート・インポート
- **自動エクスポート**: データ変更後、30秒ごとに `client-dashboard-data.json` が自動的にダウンロードフォルダに保存されます
- **手動インポート**: 「📥 インポート」ボタンをクリックして、以前エクスポートしたJSONファイルを選択
